# superstore-sales-analysis

## Project Overview
This project analyzes retail sales data from a Superstore dataset using Microsoft Excel and SQL to uncover trends in sales, profitability, customer segments, product categories, and regional performance.
The objective was to transform raw sales data into meaningful business insights through data cleaning, exploratory analysis, SQL queries, pivot tables, and an interactive Excel dashboard.

## Business Problem
Retail businesses generate thousands of sales transactions every year, making it difficult to identify which products, regions, and customer segments contribute most to revenue and profit.
This project aims to answer key business questions such as:
- Which customer segment generates the highest sales?
- Which region is the most profitable?
- Which product categories perform best?
- How do sales change throughout the year?
- Which products require management attention?

 ## Tools Used
- Microsoft Excel
- MySQL
- Pivot Tables
- Pivot Charts
- Excel Dashboard

- ## Analysis Process
The analysis was completed in the following stages:

### 1. Data Preparation
- Imported the Superstore dataset into Microsoft Excel.
- Reviewed the dataset for missing values and inconsistencies.
- Prepared the data for analysis using Excel features.

### 2. Exploratory Data Analysis (EDA)
- Analyzed sales, profit, quantity, and customer segments.
- Compared regional and category performance.
- Identified monthly sales trends.

### 3. SQL Analysis
- Imported the dataset into MySQL.
- Wrote SQL queries to answer business questions.
- Used aggregation functions, filtering, sorting, and grouping to generate insights.

### 4. Dashboard Development
- Built an interactive Excel dashboard using Pivot Tables, Pivot Charts, and slicers.
- Created KPIs to monitor overall business performance.

- ## Key Business Insights
### Overall Performance
- Total Sales: **$2,297,200.86**

### Customer Segments
- Consumer generated the highest sales (**$1,161,401.34**).
- Corporate followed with **$706,146.37**.
- Home Office generated **$429,653.15**.

### Regional Performance
- West recorded the highest profit (**$108,418.45**).
- Central generated the lowest profit (**$39,706.36**).

### Monthly Sales Trend
- November recorded the highest sales (**$349,120.07**).
- December ranked second (**$332,177.16**).
- February recorded the lowest sales (**$60,172.65**).

### Product Performance
- Phones generated the highest sales (**$330,007.05**).
- Chairs ranked second (**$328,449.10**).
- Fasteners generated the lowest sales (**$3,024.28**).

- ## Business Recommendations
Based on the analysis, the following recommendations were identified:
- Increase inventory and marketing efforts for high-performing products such as Phones and Chairs.
- Investigate the Central region to identify factors contributing to lower profitability.
- Prepare inventory ahead of peak sales periods, particularly November and December.
- Develop targeted campaigns to improve sales during slower months such as February.
- Focus on retaining Consumer segment customers while identifying opportunities to grow Corporate and Home Office sales.
